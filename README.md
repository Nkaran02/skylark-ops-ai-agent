# Skylark Ops AI Agent Dashboard

## Overview

Skylark Ops AI Agent is a web-based operational assistant dashboard designed to help with drone fleet and pilot operations using predefined AI-driven queries. The application focuses on simplifying operational decision-making by allowing users to trigger common operational workflows directly from the dashboard.

This project is currently in the **development phase**, and the system is optimized to work with a fixed set of predefined operational queries to ensure reliable and consistent responses.

## How to Use the Dashboard

To ensure the application returns data correctly, please use the following **predefined operational queries** available in the dashboard:

- **Show available pilots**  
- **Available drones**  
- **Check conflicts**  
- **Urgent reassignment**  
- **Update pilot Arjun busy**  
- **Assign pilot**  

These predefined queries are connected to the current data and logic in the system. Using custom or free-form queries may not return results at this stage.
